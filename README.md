# Gestor de Notas CLI

Gestor de notas interactivo para la terminal, construido con TypeScript

## Instalación y ejecucion

```
npm install
npm run dev
```

## Uso

Al iniciar el programa verás un menú con las siguientes opciones:

```
1. Agregar nota
2. Ver notas
3. Buscar nota
4. Eliminar nota
5. Salir
```
